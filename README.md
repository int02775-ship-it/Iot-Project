Το παρόν έργο αφορά την ανάπτυξη μιας ολοκληρωμένης εφαρμογής Διαδικτύου των Αντικειμένων (Internet of Things – IoT) με στόχο την παρακολούθηση περιβαλλοντολογικών δεδομένων σε πραγματικό χρόνο. Βασίζεται στην υλοποίηση ενός έξυπνου συστήματος παρακολούθησης που συλλέγει, αποστέλλει και απεικονίζει δεδομένα θερμοκρασίας, υγρασίας και κίνησης, ενώ παρέχει δυνατότητα απομακρυσμένου ελέγχου μέσω μιας διαδικτυακής πλατφόρμας.
